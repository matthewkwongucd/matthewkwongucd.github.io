Preventing Cyberbullying:

Report

Report cyberbullying to online service providers. Block users and report users to the social media site so they can take action against the bully.

In cases of threats of violence, child pornography, stalking and hate crimes, consider reporting this crime to law enforcement.

Talk

Talk and speak up! Let others know what's going and ask for help. Ask questions to learn what's happening.

"How it started?", "When did it start?" and "Who is involved?" are great questions to ask.

Support

If you or someone else is getting bullied, ask for help. Talk to trusted peers, mentors, and adults that can help intervene publically to over come this challenge.
